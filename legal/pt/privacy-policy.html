Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo Medi (“o App”), disponível para dispositivos iOS e Android, coleta, utiliza, armazena e protege os dados fornecidos pelos usuários. Este documento está em conformidade com padrões internacionais de proteção de dados, incluindo GDPR, princípios ARCO, CCPA e boas práticas da App Store / Google Play.

1. Responsável pelo Tratamento

O tratamento dos dados é realizado pela equipe de desenvolvimento responsável pelo Medi. Você pode entrar em contato pelo e-mail:
soporte@nexusecosystem.com

Até a constituição formal da entidade comercial, o tratamento é realizado pelos desenvolvedores responsáveis pelo App. Isso não afeta os direitos do usuário nem as medidas de privacidade aqui estabelecidas.

2. Dados que Coletamos

O Medi NÃO coleta dados pessoais sensíveis, médicos, clínicos, biométricos ou financeiros. Todas as informações permanecem armazenadas localmente no dispositivo do usuário.

Dados armazenados no dispositivo:

  • Medicamentos cadastrados pelo usuário
  • Lembretes e horários configurados
  • Histórico de tomadas (apenas local)
  • Preferências de idioma, notificações e aparência

O Medi não transmite, vende ou compartilha informações com terceiros.

3. Funcionalidade do App

Os dados são utilizados exclusivamente para:

  • Gerenciar lembretes personalizados
  • Organizar informações de medicamentos inseridos
  • Exibir estatísticas locais de uso
  • Melhorar e personalizar a experiência dentro do App

Os dados não são utilizados para publicidade, criação de perfis, análises externas ou fins comerciais.

4. Armazenamento em iOS e Android

No iOS, o App utiliza:

  • Armazenamento local seguro (Sandbox)
  • UserDefaults e arquivos criptografados pelo iOS
  • Keychain quando é necessário persistir credenciais

No Android, o App utiliza:

  • SharedPreferences seguras
  • Room Database (somente local)
  • EncryptedSharedPreferences quando o dispositivo permite

Em nenhuma plataforma os dados são enviados para servidores externos.

5. Permissões Solicitadas (iOS + Android)

5.1 Notificações

Solicitadas para permitir lembretes de medicação. O usuário pode desativá-las a qualquer momento.

5.2 Câmera

Utilizada exclusivamente para leitura de códigos de barras. As imagens não são armazenadas nem enviadas a servidores.

5.3 Alarmes Exatos (Android)

Utilizados para lembretes precisos no Android.

5.4 Tarefas em Segundo Plano (iOS)

Permitem a entrega pontual de notificações programadas.

6. Direitos do Usuário (ARCO / GDPR / CCPA)

O usuário pode exercer os seguintes direitos:

  • Acesso — Visualizar as informações armazenadas.
  • Retificação — Modificar medicamentos ou configurações.
  • Exclusão — Apagar dados ou desinstalar o App.
  • Oposição — Desativar funcionalidades que não deseja utilizar.
  • Portabilidade — Exportar o histórico em PDF (se aplicável).

Para exercer qualquer direito, entre em contato: soporte@nexusecosystem.com

7. Transferência de Dados

O Medi NÃO compartilha, transfere ou vende dados dos usuários a terceiros.

8. Segurança

  • Criptografia do sistema de arquivos
  • Isolamento por aplicativo (Sandbox)
  • Restrições em nível de hardware
  • Proteção de dados em segundo plano

10. Contato
